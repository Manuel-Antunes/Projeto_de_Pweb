# Smart Plantations

## This project uses

### Protocols
* MQTT
* HTTP
* WS

### Architeture
MVC

### Environment
* SQLDBs
* Firebase

### Tools
* Express
* Mosca
* Socket.io
* Handlebars

## How to Use
* yarn / npm install: to setup the project
* yarn / npx sequelize db:create: to create the database
* yarn / npx sequelize db:migrate: to migrate all schemas
* yarn dev: to run
